## 十二、计算规则（全部实时，DB 只存 Holding）

```
市值        = quantity × current_price
收益金额    = 当前市值 − 总成本
收益率      = 收益金额 / 总成本
财富总值    = Σ 每个 Holding 当前市值
今日收益    = Σ 每个 Holding 今日盈亏
```
数据库**不保存任何计算结果**。

> **今日收益补充（用户 22:27）**：「后端 batch 当前无『今日盈亏』字段。本地的数据是有成本跟份额的接口，可以获取当前价值完全能算出今日的亏损。」→ 今日收益由前端用 `change_percent` × 当前市值（或持仓）计算，不依赖后端新增字段。
